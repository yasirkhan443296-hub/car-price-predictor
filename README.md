# car-price-predictor
Car price prediction ML model built with Python and Scikit-learn, deployed as a web app using Streamlit. Predicts used car prices based on name, company, year, kms driven and fuel type.
# Car Price Predictor 🚗

A machine learning web app that predicts used car prices in India.

## About
Predicts the price of a used car based on:
- Car Name
- Company
- Year
- KMs Driven
- Fuel Type

## Tools Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## Model
- Algorithm: Linear Regression
- R2 Score: 78%
- Dataset: Quikr Cars India
